# Referral Bro - Product Roadmap

## Current Version: v1.0 ✅
- [x] Import contacts (CSV/JSON)
- [x] Export contacts as CSV
- [x] Quick Add form
- [x] Overlay badges on LinkedIn, Naukri, Indeed, Glassdoor, Wellfound
- [x] Toggle overlay on/off
- [x] Manual company search
- [x] AI prompt for data conversion

---

## v1.1 - Data Management
- [ ] **Import Mode Selection**: Choose between "Replace All" or "Merge & Add" when importing
- [ ] **Duplicate Detection**: Smart merge to avoid duplicate contacts
- [ ] **Delete Individual Contacts**: Remove specific entries from popup

---

## v1.2 - Messaging Templates (Released) ✅
- [x] **Custom DM Template**: Create personalized referral request message
- [x] **Direct Insertion**: Insert message directly into LinkedIn chat
- [x] **Template Variables**: Auto-insert `[HR Name]`, `[Company Name]`, `[Position]`
- [x] **Multiple Templates**: Saved templates for "Referral" and "Interview" requests
- [x] **Overlay UI**: "Ask Referral" buttons directly in the easy-access overlay


---

## v2.0 - Enhanced Features
- [ ] **LinkedIn Profile Detection**: Show badge on company employee profiles
- [ ] **Auto-detect contacts from LinkedIn**: Scan connections and auto-import
- [ ] **Notes editing**: Edit notes directly in the popup
- [ ] **Search history**: Recently viewed referral matches
- [ ] **Statistics**: Track how many referrals you've used

---

- Browser sync across devices
- Share contact lists (encrypted)
